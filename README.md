# BlogSpot - Modern Blog WebsiteA complete modern blog website with admin panel, built with clean frontend and backend separation.## 🚀 Features### Frontend- **Modern Design**: Clean, responsive UI with smooth animations- **Blog System**: Browse and read blog posts with comments- **User Interaction**: Comment system with real-time updates  - **Newsletter**: Subscribe to newsletter with email validation- **Support System**: Contact forms and FAQ section- **Career Portal**: Job listings with application system- **Admin Panel**: Complete admin dashboard with CRUD operations### Backend- **RESTful API**: Clean API structure with Express.js- **Authentication**: Admin login system- **Blog Management**: CRUD operations for blog posts- **Comment System**: Moderation and management- **User Management**: Admin user controls- **Support Tickets**: Ticket management system- **Career Applications**: Job application handling## 📁 Project Structure```blog/├── frontend/               # Frontend **Frontend**: http://localhost:3000- **Backend API**: http://localhost:5000- **Admin Panel**: http://localhost:3000/pages/admin.html- **Support Page**: http://localhost:3000/pages/support.html- **Careers Page**: http://localhost:3000/pages/careers.html## 🔧 API Endpoints### Authentication- `POST /api/auth/login` - Admin login- `POST /api/auth/logout` - Admin logout- `GET /api/auth/verify` - Verify token### Blogs- `GET /api/blogs` - Get all blogs- `GET /api/blogs/:id` - Get single blog- `POST /api/blogs` - Create blog- `PUT /api/blogs/:id` - Update blog- `DELETE /api/blogs/:id` - Delete blog### Comments- `GET /api/comments` - Get comments- `POST /api/comments` - Add comment- `PUT /api/comments/:id/approve` - Approve comment- `DELETE /api/comments/:id` - Delete comment### Users- `GET /api/users` - Get all users- `GET /api/users/:id` - Get single user- `PUT /api/users/:id` - Update user- `DELETE /api/users/:id` - Delete user### Support- `GET /api/support/tickets` - Get tickets- `POST /api/support/tickets` - Create ticket- `PUT /api/support/tickets/:id` - Update ticket### Careers- `GET /api/careers/jobs` - Get job listings- `GET /api/careers/jobs/:id` - Get single job- `POST /api/careers/apply` - Submit application- `GET /api/careers/applications` - Get applications## 👨‍💻 Admin Credentials**Username**: `admin`  **Password**: `admin123`## 🎨 Customization### Styling- Edit
